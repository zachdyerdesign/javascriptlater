ZDD

Overview

ZDD is a website templating engine designed to run on free web hosts so you can easily manage your website with free web hosting such as Github Pages and Google Drive.

Instructions

  1. Download the files from https://github.com/zachdyerdesign/zdd
  2. Go to modules to update module data and create new modules.
  3. Use the zdd functions inside of script tags anywhere in your html to add dynamic menus and more.

Templating Functions

  1. zdd.getMenu(name) - Use this to get use your dynamic menu stored in zdd.data.menus
  2. zdd.getFooter() - Use this to display your footer information stored in zdd.data.footer
  3. zdd.getIcon(icon, size) - Use this to display an icon from font awesome. Find more icons at http://fortawesome.github.io/Font-Awesome/icons/
