JavaScriplater

Overview

JavaScriplater is a website templating engine designed to run on free web hosts so you can easily manage your website with free web hosting such as Github Pages and Google Drive.

Instructions

  1. Download the files from https://github.com/zachdyerdesign/scriptlater
  2. Go to data/data.js to update general, menu, and footer information.
  3. Use the scriptlater functions inside of script tags anywhere in your html to add dynamic menus and more.

Templating Functions

  1. scriptlater.getMenu(name) - Use this to get use your dynamic menu stored in scriptlater.data.menus
  2. scriptlater.getFooter() - Use this to display your footer information stored in scriptlater.data.footer
  3. scriptlater.getIcon(icon, size) - Use this to display an icon from font awesome. Find more icons at http://fortawesome.github.io/Font-Awesome/icons/
